Шаблон приложения для React

- Разбиение бандла на части
- Есть анализатор бандла
- кастомный еслинт конфиг

### Генератор компонентов

Встроен генератор компонентов реакта. Это обеспечит единство стиля оформления и ускорит создание компонентов.
Соответствующие шаблоны можно найти в папке `/templates/`.


1) Создание обычного компонента.
```bash
pnpm grc MyComponent
```
Эта команда создаст пустой компонент с названием MyComponent в директории components.

2) Создание контекста.
```bash
pnpm grc App --type=context
```
Команда сгенерирует контекстный компонент и положит его в папку contexts.
