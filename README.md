# ReactAppTemplate
Шаблон приложения для React


## Фичи

- Разбиение бандла на части
- Есть анализатор бандла
- кастомный еслинт конфиг

### Генератор компонентов

Встроен генератор компонентов для обеспечения скорости создания компонентов и единства стиля. 

Использование:
```bash
pnpm grc component MyComponent
```
Эта команда создаст пустой компонент с названием MyComponent в директории components.
